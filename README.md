# Escape the Scroll 📱🛑

## About the Project
**"Escape the Scroll"** (also known as *Weapons of Mass Distraction*) is a psychological web application built for a hackathon. The core goal of this project is to raise awareness about the addictive nature of endless scrolling on social media platforms and its impact on our mental health. 

The application creatively simulates the experience of being "trapped" in a social media feed. By using psychological hooks (like fake notifications and infinite scrolling), it demonstrates how easily we lose track of time—before forcefully breaking the cycle and guiding the user into a distraction-free "Focus Mode."

## Key Features ✨
- **Landing Page**: Introduces the project's motive—taking control of your scroll and being mindful of content consumption.
- **The Fake Feed ("TikTik")**: A simulated infinite-scrolling social media feed that acts as a "trap." After a certain amount of scrolling, a trap screen is triggered, alerting the user to their mindless scrolling behavior.
- **Focus Mode**: Following the trap, the user is redirected to a calm, minimalist environment designed to promote productivity and focus, breaking the cycle of distraction.
- **Interactive UI**: Includes engaging elements, structured navigation across all pages, and a cohesive dark-themed style.

## Technologies Used 🛠️
- **HTML5**: For semantic structuring of the application.
- **Vanilla CSS3**: Custom styles containing a unique theme, responsive design, and modern layouts.
- **Vanilla JavaScript**: Powers the feed logic, infinite scrolling simulation, time-tracking trap mechanism, and interactive user flows.

## How to Run the Project 🚀
1. Clone or download the source code to your local machine.
2. The project uses standard web technologies, so no build tools or package managers are required.
3. Simply open `index.html` in any modern web browser to start the experience.
4. Click **"start scrolling"** or navigate to the **"TikTik"** tab to experience the feed and trap mechanism.
